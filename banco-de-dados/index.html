Sobre a Disciplina

Banco de Dados é uma disciplina que ensina os conceitos básicos e as práticas essenciais para a criação, manipulação e gerenciamento de dados. O objetivo é fornecer uma base sólida para o entendimento de como os dados são organizados, armazenados e acessados.

Durante o curso, os alunos explorarão uma série de tópicos essenciais, começando com a introdução aos fundamentos dos bancos de dados, onde serão abordados os diferentes tipos de bancos de dados e modelos de dados. Em seguida, o curso se aprofundará no modelo relacional, que inclui a estruturação de tabelas, definição de chaves primárias e estrangeiras.

Os alunos também aprenderão sobre a linguagem SQL (Structured Query Language), incluindo comandos DDL (Data Definition Language) para a definição de estruturas de banco de dados, comandos DML (Data Manipulation Language) para manipulação de dados e comandos DCL (Data Control Language) para controle de acesso e permissões.

O curso também abordará a criação e uso de procedimentos armazenados e gatilhos (triggers), técnicas para indexação e otimização de consultas para melhorar o desempenho.

Ao concluir a disciplina, os alunos estarão aptos a projetar, implementar e gerenciar bancos de dados de forma eficiente e segura, utilizando as melhores práticas e ferramentas disponíveis.
